This is a MERN Stack Hospital Management System that has 3 main parts: 
 - Frontend
 - Backend
 - Dashboard

Frontend features:
 - Dashboard
 - Add Admin
 - Add Doctor
 - Add Receptionists
 - See all doctors
 - See all appointments
 - Manage appointments
